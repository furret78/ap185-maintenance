Actual boss data table calculation:
MENU_STATIC_ADDRESS + (0x267c0 + arg1 * 0x1d + arg2) (offset)
The IDs in the Python file count from 0. Need to +2 before checking the data.
From Tsukasa onwards, that's +1.
If Chimata, +0. If Nitori, +11. If Takane, +3.

arg2 = Boss ID
